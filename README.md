#AI Resume & Portfolio Builder

An intelligent, Generative AI-powered web application that transforms raw student data into professional, ATS-friendly resumes, cover letters, and portfolio biographies. This project was developed as part of the AICTE Edunet Foundation & IBM SkillsBuild Internship.

#Key Features

Generative AI Integration: Uses Google Gemini 1.5 Flash to dynamically tailor content based on specific job descriptions.

Live Preview: Instantly visualize changes in a professional layout as you input your data.

Multiple Design Templates: Switch between Classic, Modern, and Minimalist styles.

Professional Exports: Download your results instantly in PDF, MS Word (.docx), or Plain Text formats.

ATS Optimization: The AI engine identifies and incorporates relevant keywords from job descriptions to improve hiring chances.

#Tech Stack

Frontend: Streamlit

AI Engine: Google Generative AI (Gemini API)

Document Logic: FPDF (PDF) & python-docx (Word)

Language: Python 3.x
